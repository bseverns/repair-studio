# Triage Scoring Rubric (0–10)

- **Impact (0–3):** safety/resilience/user harm reduced.
- **Reversibility (0–2):** easy to undo if needed.
- **Time (0–2):** ≤30m =2, ≤90m =1, >90m =0.
- **Learning value (0–2):** teaches the group something reusable.
- **Dependency unblock (0–1):** others are waiting.

Highest score first unless safety dictates otherwise.
