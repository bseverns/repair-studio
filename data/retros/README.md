# Retro Logs

Every session deserves a quick look back. Store the completed retro notes in
this directory so we can see how the system evolves over time.

## How to record a retro

1. Copy `templates/retro-template.md` into this folder and rename it
   `YYYY-MM-DD-retro.md`.
2. Capture the conversation immediately after the session wraps while
   memories are fresh.
3. Reference the latest session notes and metrics entry when you fill in
   conditions and evidence.
