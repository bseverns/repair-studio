# Manuals

Drop reference manuals here, organized by category:

- `machines/` — hardware, tools, printers, cutters
- `software/` — applications, firmware, drivers
- `instruments/` — rigs, synths, controllers
- `spaces/` — room AV, safety, building systems
- `host-briefing-escalation.md` — 5-minute huddle script that keeps signage,
  policies, and on-call contacts aligned before each session.

> After adding files, run `python3 scripts/build_manuals_index.py` to regenerate
> `INDEX.md`.
