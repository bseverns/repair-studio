# Manuals

Drop reference manuals here, organized by category:

- `machines/` — hardware, tools, printers, cutters
- `software/` — applications, firmware, drivers
- `instruments/` — rigs, synths, controllers
- `spaces/` — room AV, safety, building systems

> After adding files, run `python3 scripts/build_manuals_index.py` to regenerate `INDEX.md`.
